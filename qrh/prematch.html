G301 -- Be prompt.
Description A DRIVE TEAM member may not cause significant delays to the start of their MATCH. Delay occurs when the scheduled start time has passed and the team, with ARENA access, is not ready or making a good-faith effort to be ready.
Penalty Verbal Warning Major Foul if subsequent within tournament phase Robot Disabled if not ready within 2 minutes and no good faith effort
Extra Consult the manual for details on Qualification vs. Playoff handling.
G302 -- Limit what you bring to the FIELD.
Description Items brought to the FIELD, beyond the ROBOT and OPERATOR CONSOLE, must fit in the ALLIANCE AREA, be worn/held by DRIVE TEAM, or be an accommodation item. Items must not create hazards, exceed 6'6", communicate externally, block visibility, or interfere with the ARENA.
Penalty Match will not start until situation is remedied Yellow Card if discovered or used inappropriately during MATCH
G303 -- ROBOTS on the FIELD must come ready to play.
Description ROBOTS must be safe, inspected, compliant with any modifications, the only team-provided item on the FIELD, show correct ALLIANCE color, and remain motionless after OpMode initialization.
Penalty Match will not start until requirements met (if quick remedy) Disabled if not quick remedy; Red Card if uninspected or noncompliant ROBOT participates
Extra Head REFEREE may require re-inspection before next MATCH.
G304 -- ROBOTS must be set up correctly on the FIELD.
Description ROBOTS must start over a LAUNCH LINE, touching their GOAL or FIELD perimeter, fully contained on their side, not entangled or attached, in STARTING CONFIGURATION, and within allowed preload limits.
Penalty Match will not start until setup corrected (if quick remedy) Disabled if not a quick remedy
Extra ROBOTS must be fully contained within FIELD perimeter wall.
G305 -- Teams must select an OpMode.
Description Teams must select and initialize an OpMode on the DRIVER STATION app before the MATCH. AUTO OpModes must have the 30-second AUTO timer enabled.
Penalty Match will not start until situation remedied Disabled if OpMode cannot initialize or issue not quickly fixed
